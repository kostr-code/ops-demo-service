# ops-demo-service
Сервис представляет собой тестовое клиент-серверное приложение.

Запуск сервера производится командой python main_server.py
Запуск клиента производится командой python main_client.py

Сервер:
	* Обрабатывает запросы от клиентов (Посчиать сумму чисел, вывести сроку, сделать любую из команд с задержкой 1, 5, 10 секунд)
	* Работает в многопоточном режиме, чтобы клиенты могли параллельно обращатсья к серверу
	* Используемые стек: python, IPC (sockets)

Клиент:
	* Требует вести ID сессии
	* Может выполнить одну из команд
		** Запросить сумму чисел
		** Запросить случайную строку
		** Получить ответ от сервера с задержкой

Необходим для отработки DevOps навыков и умений многопоточной разработк на python + немного познакомиться с IPC на python
